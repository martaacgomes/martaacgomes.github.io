Regras do Jogo:

  1. Podem jogar até 5 participantes em simultâneo
  2. Cada jogador deve escolher a personagem com a qual vai jogar
  3. O jogador tem de escolher uma carta do baralho das "perguntas" e entregar a outro jogador para que este leia a questão
  4. Se o jogador acertar a resposta à questão, este pode lançar os dados e avançar quantas casas quantos pontos tiverem as faces dos dados voltadas para cima
  5. Se o jogador calhar numa casa em que começa uma escada, deve avançar para o a casa que está no final da mesma
  6. Se o jogador calhar uma casa que contém a cara de uma cobra, deve recuar para a casa em que está a cauda
